Рады представить вам второй бета-релиз сайта, который привносит собой бета-версию системы гильдий, огромное количество изменений, исправлений и некоторые мини фишки.
## Нововведения:
* Добавлено API получения конкретного участника гильдии.
* Добавлено API для создания, получения и управления системой заявок в гильдию.
* Добавлен обработчик ошибок в API.
* Добавлены загрузочные скелеты в те места, где требуется загрузка данных.
* Добавлен beta. домен, где тестируются около работающие дев билды сайта.
* Добавлена обработчик ошибок на случай, когда API статистики недоступно.
* Добавлен следующий функционал системы гильдий: просмотр существующих гильдий, отправка заявки в гильдию, управление данными гильдии, управление игроками гильдии, управление заявками гильдии.
## Экспериментальные нововведения (доступны только на нескольких страницах для теста):
* Добавлены оповещения в верхней части экрана, когда происходит успешное или неудачное событие (доступно на странице управления гильдией и участниками)
* Добавлен более простой способ авторизации при обращении к API с помощью заголовка.
## Изменения:
* Футер сайта был доработан, там появились заготовки под юридические документы и обновлён дизайн копирайта и версии сайта.
* Код авторизации был значительно улучшен, логика авторизации стала проще и быстрее.
* Оптимизирован размер кода всех ранее добавленных страниц и API.
* Улучшена адаптация сайта под мобильные устройства.
* Улучшен код валидации данных.
* Изменены ссылки на аватар игрока.
## Исправления:
* Различные исправления в работе API.
* Исправил обновление кнопки ЛК при выходе из аккаунта.
* Исправлено получение данных API в тех местах, где возможен только 1 ответ, а не массив.
* Исправлен размер навигации и футера.
* Исправлено отображение изображений на beta. домене.
