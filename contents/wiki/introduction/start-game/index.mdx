## 1. Зарегистрируйтесь на Minecraft сервере
Для этого зайдите по нашему IP - `mc.foxworld.ru:25566` и пройдите регистрацию.

Это нужно, чтобы вы смогли в личный кабинет на сайте.
## 2. Заполните заявку на сайте
Нажмите по вашему скину в правом верхнем углу и нажмите на ссылку "Заполните анкету"

Далее заполняете анкету, отправляете и ожидаете ответа в течение часа.
## 3. Зайдите на Discord сервер
Для игры на нашем сервере нужно быть участником нашего сервера.

Помимо этого вся основная аудитория общается именно там, а ещё там самые свежие новости)
## 4. Свяжите Discord и Minecraft аккаунты
Зайдите на Minecraft сервер, вас кикнет с причиной привязки и покажет в этом сообщении код.

Отправьте этот код нашему боту @FoxMinecraft#4137 в лс.

* Java Edition: 1.21 - 1.21.4
