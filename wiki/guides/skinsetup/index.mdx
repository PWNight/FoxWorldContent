## Какой плагин мы используем?
Наш сервер использует плагин [SkinRestorer](https://www.spigotmc.org/resources/skinsrestorer.2124/) для управления скинами игроков.

Если вам интересно, то вы можете ознакомиться с [документацией плагина](https://skinsrestorer.net/docs)

## Способы установки скинов
Всего есть 3 способа установить скин.
### 1. Установка скина через /skins
Эта команда открывает вам меню с набором уже готовых скинов.

Не является особо распространённым способом.
### 2. Установка по никнейму
Для этого вам нужно найти лицензионный аккаунт с понравившимся вам скином.
<br/>Найти свой скин по нику вы можете, например, [на NameMC](https://namemc.com/).

Как только найдёте подходящий ник, используйте команду `/skin set [ник]`
### 3. Установка по ссылке
Если у вас есть файл с вашим скином, то вы можете установить скин по ссылке на ваш файл.

Для этого способа следуйте следующим шагам:
1. Перейдите на [https://imgur.com/upload](https://imgur.com/upload).
2. Загрузите туда ваш скин.
3. Перейдите по полученной ссылке.
4. Нажмите ПКМ по вашему скину и выберите "Скопировать ссылку на изображение".
5. Зайдите на сервер и используйте команду `/skin set "[ссылка]"`. Не забудьте про кавычки!
